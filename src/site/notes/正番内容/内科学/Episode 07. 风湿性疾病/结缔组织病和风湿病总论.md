# 概述
+ 风湿性疾病是各种病因引起的一组累及骨、关节和关节周围软组织，（包括肌肉、肌腱、韧带、筋膜、滑膜、神经等）及其他相关组织和器官的慢性疾病
+ 结缔组织病是风湿性疾病的一大类
+ 特点
	+ *慢性*病程，常累及*肌肉关节*
	+ 属于*自身免疫病*
	+ 以血管和结缔组织慢性炎症为*病理*基础
	+ 累及*多个系统*
	+ 对*激素治疗*有一定反应
	+ 多有*免疫球蛋白、ESR、CRP 升高*
	+ 同一疾病，临床谱及预后*差异大*

# 辅助检查
## 自身抗体检测
1. 抗核抗体 ANAs：ANA 阳性==最常见于==[[正番内容/内科学/Episode 07. 风湿性疾病/系统性红斑狼疮\|系统性红斑狼疮]]、结缔组织病，可做筛查
	1. 抗 DNA 抗体：*==抗 ds-DNA 抗体==*-->[[DLC/en-US/SLE\|SLE]] 活动性
	2. 抗组蛋白抗体-->药物性狼疮、[[DLC/en-US/SLE\|SLE]]
	3. 抗非组蛋白抗体-->抗 ENA 抗体
		1. *==抗 Sm 抗体==*-->[[DLC/en-US/SLE\|SLE]] 诊断
		2. *==抗 SS-A/SS-B 抗体==*-->干燥综合征、[[DLC/en-US/SLE\|SLE]]
		3. *==抗 U1-RNP 抗体==*-->与雷诺现象、肌炎相关
		4. *==抗 rRNP 抗体==*--> [[DLC/en-US/SLE\|SLE]]（神经精神狼疮）
		5. 抗 Scl-70 抗体-->硬皮病
		6. 抗着丝点抗体-->硬皮病
		7. 抗 Jo-1 抗体-->皮肌炎
	4. 抗核仁抗体
		1. 抗核仁抗体
		2. 抗心磷脂抗体
		3. 抗β2-GPI 抗体
2. 类风湿因子 RF：见于 [[DLC/en-US/RA\|RA]]（80%）、干燥综合征、[[DLC/en-US/SLE\|SLE]]
3. ANCA：抗中性粒细胞胞浆抗体，见于血管炎病（Wegener 肉芽肿）
4. 抗磷脂抗体：包括抗心磷脂抗体、狼疮抗凝物、抗β2-GPI 抗体，与血小板减少、动静脉血栓、习惯性自发性流产有关
5. 抗角蛋白抗体谱
	1. 抗核周因子抗体
	2. 抗角蛋白抗体
	3. 抗环瓜氨酸多肽抗体（CCP）-->诊断[[正番内容/内科学/Episode 07. 风湿性疾病/类风湿性关节炎\|类风湿性关节炎]]特异性最高
## 人类白细胞抗原检测
1. HLA-B27-->脊柱关节病，强直性脊柱炎
2. HLA-B5-->白塞病
3. HLA-DR4-->类风湿关节炎
## 补体
+ CH50、C3、C4 有助于 [[DLC/en-US/SLE\|SLE]] 和血管炎的诊断及疾病活动性、疗效判断
# 诊断
+ 病史+体检
+ 特殊检查：
	+ 关节镜和关节滑液
	+ `自身抗体`
	+ 补体
	+ 病理
	+ 影像
# 治疗
1. 非甾体类抗炎药：改善症状，不能根治
2. [[未发售DLC/糖皮质激素\|糖皮质激素]]：改善症状最有效的药物，一线药物，不能根治
3. 改变病情抗风湿药DMARDs
4. 生物制剂
