# 概述
+ 结核杆菌引起的慢性感染性疾病
+ 最常见：原发型肺结核
+ 最严重：结核性脑膜炎
+ 传染源：开放性肺结核，呼吸道为主要传播途径
# 诊断
1. 病史
	1. 中毒症状：长期低热、轻咳、盗汗、乏力、食欲减退、消瘦等
	2. 结核病接触史
	3. 接种史：卡介苗
	4. 有无急性传染病史：特别是麻疹、百日咳等可是机体免疫功能暂时降低
	5. 有无结核过敏表现：结节性红斑、疱疹性结膜炎等
2. 结核菌素试验
	1. 小儿受结核感染 4-8 w 后即可呈现阳性反应
	2. 结核菌素试验：迟发型变态反应
	3. 临床意义
		1. 阳性：
			1. 接种卡介苗后
			2. 年长儿无明显临床症状，仅呈一般阳性反应：曾感染过结核
			3. 婴幼儿，尤其是未接种卡介苗者呈阳性：体内有新的结核病灶，年龄越小，活动性结核可能性越大
			4. 强阳性：活动性结核
			5. 阴转阳或反应强度由<10mm-->>10 mm 且增幅超过 6 mm：近期有感染
		2. 阴性：
			1. 未感染过
			2. 结核迟发型变态反应前期（初次感染 4-8 w 内）
			3. 假阴性反应：机体免疫低下或受抑制
			4. 技术误差或结核菌素失效
3. 实验室检查：
	1. 结核分枝杆菌检查：确诊
	2. 免疫学及分子生物学：ELISA、r-干扰素释放实验
	3. 血沉：增快-->活动性
4. 影像学：
	1. ==X 线==
	2. CT：肺结核的诊断和鉴别诊断
5. 其他：
	1. 纤支镜
	2. 周围淋巴结穿刺液涂片检查
	3. 肺穿刺活体组织检查或胸腔镜取肺活体组织检查
# 治疗
1. 一般治疗
	1. 注意营养，高蛋白高维生素食物
	2. 卧床休息
	3. 避免传染麻疹、百日咳等疾病
2. 药物治疗：
	1. 目的-->杀灭病灶中的结核分枝杆菌；防止血行播散
	2. 原则：早期、联合、全程、适量、规律、分段
	3. 杀菌药：异烟肼、利福平、链霉素、吡嗪酰胺
	4. 抑菌药：乙胺丁醇
3. 治疗方案：
	1. 标准疗法
	2. 两阶段疗法：强化治疗阶段；巩固治疗阶段
	3. 短程疗法
# 预防
1. 控制传染源
2. 普及卡介苗接种
	1. 禁忌症：
3. 预防性抗结核治疗
	1. 目的：预防儿童活动性肺结核；预防肺外结核；预防青春期结核复燃
	2. 适应症
# 原发性肺结核
## 概述
1. 小儿肺结核主要类型
2. 包括原发综合征和支气管淋巴结结核，前者由非原发病灶、局部淋巴结病变和两者相连的淋巴管炎组成
## 病理
1. 肺原发病灶多位于右侧，肺上叶底部和下叶的上部，近胸膜处
2. 基本病变为渗出、增殖、坏死
3. 典型的原发综合征呈“双极”病变，即一端为原发病灶，一端为肿大的肺门淋巴结、纵隔淋巴结
## 临床表现
1. 症状轻重不一，轻者可无症状
2. 一般起病缓慢，有低热、盗汗、纳差、乏力等结核中毒症状
3. 最常见：干咳和轻度呼吸困难
4. 婴儿可表现为体重不增或生长发育障碍
5. 体格检查：周围淋巴结不同程度肿大；肺部体征可不明显，与肺内病变不一致，婴儿可伴肝大
## 诊断
1. 原发综合征：肺内实质浸润伴肺门淋巴结和纵隔淋巴结肿大，X 线典型哑铃状双极影
2. 支气管淋巴结结核：炎症型、结节型、微小型
## 检查
1. ==X 线==，CT
2. 纤支镜
## 结核性脑膜炎
## 概述
1. 最严重的类型，常在结核原发感染后 1 y 内发生，多见于 3 y 以内婴幼儿
2. 由血行播散而来
3. 与婴幼儿中枢神经发育不成熟、血脑屏障功能不完善、免疫功能低下有关
4. 结核性脑膜炎亦可由脑实质或脑膜的结核病灶溃破，结核分枝杆菌进入蛛网膜下腔及脑脊液中所致
