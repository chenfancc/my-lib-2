# 分类
1. 生理性黄疸
	1. 一般情况好
	2. 出现较晚，持续较短
	3. 进展较慢，程度较轻
2. 病理性黄疸
	1. 一般情况差
	2. 出现较早，持续较长
	3. 进展较快，程度较重
# 诊断
+ 满足任一
	+ 生后 24 h 内出现黄疸
	+ 血清胆红素足月儿>···，早产儿>···，或每日上升超过···
	+ 黄疸持续时间足月儿>2 w，早产儿>4w
	+ 黄疸退而复现
	+ 血清结合胆红素>···
# 原因
1. 胆红素生成过多
2. 肝脏胆红素代谢障碍
3. 胆汁排泄障碍
