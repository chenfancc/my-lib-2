# 肺泡表面活性物质

</div>



---
>[!note] 肺泡表面活性物质 PS
>合成：肺泡 II 型上皮细胞
>产生时间：18-20 w 开始产生，35-36 w 迅速增加并达肺成熟水平
>作用：
>+ 降低肺表面张力
>+ 防止呼气末肺泡萎缩塌陷
>+ 保持功能残气量
>+ 稳定肺泡内压
>+ 减少液体向肺泡内渗出


</div></div>

# 发病机制
1. 换气功能降低
2. 通气/血流比降低
3. 肺血管渗透性增加
4. 动脉导管卵圆孔再开放
5. 嗜伊红肺透明膜形成
# 临床表现
+ 生后不久（in 6 h）出现：
	+ 进行性加重的呼吸窘迫
	+ [[正番内容/内科学/Episode 01. 呼吸系统/呼吸衰竭\|呼吸衰竭]]
	+ 生后 24-48 h 病情最严重
	+ 72 h 后明显好转
+ 体征：
	+ 呼吸急促
	+ 发绀
	+ 呼吸困难：吸气三凹征、鼻翼扇动、呼气呻吟
	+ 胸廓扁平、呼吸音低
# 检查
+ 羊水或气管分泌物检测：==卵磷脂/鞘磷脂（L/S）比值-->判断肺成熟的标志==
+ 泡沫试验
+ X 线检查
	+ 毛玻璃样改变
	+ 支气管充气征
	+ 白肺
	+ 肺容量减少
# 鉴别诊断
+ 湿肺
+ B 组链球菌肺炎
+ 膈疝
# 治疗
1. PS 替代疗法
2. 持续呼吸道正压机械通气
3. 一般支持治疗
