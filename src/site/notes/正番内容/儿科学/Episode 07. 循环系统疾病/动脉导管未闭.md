# 分型
1. 管型
2. 漏斗型
3. 窗型
# 血流动力学特点
1. 左房左室大，主动脉结正常或突出、搏动强烈，肺循环充血
2. 周围动脉舒张压下降，体循环血不足
3. 相对性二尖瓣狭窄
4. 艾森曼格综合征
# 临床表现
1. 差异性发绀
2. 胸骨左缘上方闻及连续性机器样杂音，常伴震颤；肺动脉高压者可仅有收缩期杂音
3. 舒张压降低，脉压增大，出现周围血管征
# 治疗
1. 生后一周用吲哚美辛
2. 手术或介入
