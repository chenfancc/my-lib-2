# 概述
+ 研磨力量是造成半月板损伤的主要原因
+ 屈膝、内收、内旋、研磨
# 临床表现
+ McMurray test：外旋外翻时，伸膝诱发疼痛-->外侧半月板撕裂；内旋内翻时，伸膝诱发疼痛-->内侧半月板损伤；关节完全屈曲下有弹响-->后角损伤；伸直到 90 度弹响-->体部损伤
+ Apley test：按压状态外旋疼痛-->内侧半月板损伤；上体状态外旋疼痛-->内侧副韧带损伤
# 检查
1. X 线：排除骨性损伤
2. MRI： 确诊手段-->Stollerd 分级
	1. 0 级-->半月板内均匀信号
	2. 1 级-->点状高信号
	3. 2 级-->线状高信号
	4. 3 级-->高信号延伸至关节面
# 治疗
+ 急性期：RICE
	+ Rest、Ice、Compression、Elevation
+ 关节镜：半月板缝合或切除
