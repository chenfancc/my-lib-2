# 病因
+ 胆道梗阻：结石、蛔虫、炎性狭窄、肿瘤
+ 细菌入侵
# 临床表现
+ Charcot 三联征：寒战高热、黄疸、腹痛
+ 雷诺五联征：三联征+神志改变、低血压
+ 血 WBC 升高、肝功能损害、代谢性酸中毒
# 治疗
+ 紧急接触梗阻并引流
	+ 年龄大以及基础病选 PTCD 经皮肝穿刺置管引流
+ 非手术：抗休克、抗感染、为手术治疗作准备
