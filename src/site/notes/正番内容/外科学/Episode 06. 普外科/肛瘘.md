病因、诊断、治疗
# 概述
+ 瘘口一端通皮肤、一端通肛肠
+ 大多是直肠肛管周围脓肿的后遗症
# 分类
+ 高低位
	+ 低位肛瘘
		+ 低位单纯性：肛瘘切除术
		+ 低位复杂性：瘘管切开术
	+ 高位肛瘘
		+ 高位单纯性：挂线疗法
		+ 高位复杂性：切开+挂线
+ 分型
 ![插件/肛瘘.jpg](/img/user/%E6%8F%92%E4%BB%B6/%E8%82%9B%E7%98%98.jpg)
# 临床表现
+ 外口反复脓性、血性分泌物
+ 反复肛周感染
+ 高位肛瘘外口有气体、粪便排出
# 治疗
+ 一般治疗：抗菌、坐浴、理疗
+ 必须手术治疗
	+ 瘘管切开术
	+ 挂线疗法
	+ 肛瘘切除术
