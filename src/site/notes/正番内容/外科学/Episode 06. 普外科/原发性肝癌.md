# 病理
1. 肉眼：结节型、巨块型、弥漫型
2. 大小：
	1. 微小肝癌：<2 cm
	2. 小肝癌：(2，5]
	3. 大肝癌：(5，10]
	4. 巨大肝癌：>10 cm
3. 镜下：肝细胞型、胆管细胞型、混合型
4. 转移：
	1. 经门静脉肝内转移最常见
	2. 血行转移：肺最常见
	3. 淋巴转移：肝门、胰周、腹膜后
# 临床表现
+ 早期缺乏典型临床表现
+ 肝区疼痛
+ 肝大或右上腹肿块
+ 乏力消瘦纳差、黄疸、腹胀
# 诊断
1. AFP>400（排出妊娠、活动性肝病、生殖胚胎腺瘤）
2. 影像学：B 超、增强 CT、MR、选择性肝动脉造影
3. 肝穿刺细胞学检查
# 治疗
1. 肝切除：首选、最有效
	1. 要求：Child A、B 级，无转移
2. 肝移植：Child C 级小肝癌，或米兰标准
3. 消融治疗、TACE
4. 其他治疗
