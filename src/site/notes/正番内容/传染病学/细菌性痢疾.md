# 概述
+ 志贺菌引起的肠道传染病
+ 表现为腹泻、腹痛、里急后重及粘液脓血便，可伴有发热、全身毒血症
+ 主要病理变化：直肠和乙状结肠的炎症与溃疡
+ 严重者可出现感染性休克和或中毒性脑病
# 病原学
+ G<sup>-</sup>杆菌，兼性厌氧菌，无鞭毛，==有菌毛==
+ 抵抗力：生存力强，对理化因素抵抗力弱
+ 毒素
	+ 内毒素：各群均有
	+ 外毒素：==A 群产生志贺毒素==，具有神经毒性、细胞毒性、肠毒性作用
+ 抗原结构
	+ ==A 群==：痢疾志贺菌，==引起严重症状==
	+ ==B 群==：福氏志贺菌，主要==流行菌群==
# 流行病学
+ 传染源：病人、==带菌者==
+ 传播途径：消化道途径，水源、食物等
+ 易感人群：普遍易感，不同菌群、血清型间无交叉免疫，易反复感染
+ 流行特征：
	+ 季节性：中年散发，夏秋季发作率升高
	+ 年龄：儿童（5 岁以下）、青壮年发病率高
# 发病机制
+ 细菌突破黏膜屏障进入肠道，机会致病
+ 细菌侵袭结肠黏膜上皮细胞和固有层，并大量繁殖，导致肠粘膜炎症反应和小血管微循环障碍，肠粘膜炎症、坏死、溃疡形成，导致腹泻，粘液脓血便，腹痛，里急后重
+ 细菌释放内毒素入血引起发热和全身毒血症状，进而引起感染性休克、DIC、重要脏器功能衰竭，临床表现为中毒性菌痢
# 病理
+ 以乙状结肠和直肠病变最显著
+ 急性期病变：特征性假膜性炎，假膜脱落形成溃疡
+ 慢性期病变：瘢痕和息肉形成，少数肠腔狭窄
# 临床表现
## 急性菌痢
1. *==普通型（典型）==*：*刻板印象型*
	1. 全身毒血症状：起病急，高热可伴发冷寒战，头痛、乏力等
	2. 腹泻：先为稀水便，典型粪便为粘液脓血便，10 余次/日，便量少
	3. 腹痛：==左下腹压痛==，肠鸣音亢进
	4. 里急后重
	5. 自然病程 1-2 周，多数可痊愈，少数转为慢性
2. 轻型（非典型）：*屁事没有型*
3. ==重型==：*要拉死了型*
	1. 全身==毒血症状重==：急起发热
	2. 肠道==症状重==：腹痛，明显里急后重
	3. 腹泻==次数多==：次数>30 次/日，稀便，脓血便
	4. 后期出现严重腹胀及中毒性肠麻痹, 常伴有呕吐, 严重失水引起外周循环衰竭
	5. 多见于老年人、体弱和营养不良者
4. ==*中毒性菌痢*==：*要死了型*
	1. 2-7 y 儿童多见，起病急骤，突起畏寒发热，全身中毒症状重，可有嗜睡、昏迷、反复抽搐
	2. 局部肠道症状较轻或缺如，24 h 内可出现痢疾样粪便
	3. 按临床表现分为：
		1. 休克型：感染性休克
		2. 混合型：最凶险
		3. 脑型：脑水肿、脑疝
## 慢性菌痢
+ 菌痢反复发作或迁延不愈达==2 个月以上者==
+ 分为：慢性迁延型、急性发作型、慢性隐匿型
+ ==粪便培养阳性==
# 实验室检查
+ 一般检查
	+ 血常规
	+ 粪便常规：涂片看不到痢疾杆菌，只能看到大量的白细胞、脓细胞和红细胞
+ 病原学检查
	+ 细菌学检查
	+ 特异性核酸检测
+ 免疫学检查
	+ 早期诊断，但假阳性高，少用
# 诊断
+ 流行病学资料：多发夏秋季，有不洁饮食史，或与病人接触史
+ 临床资料：急性、慢性菌痢、中毒性菌痢的临床表现
+ 实验室检查
	+ 粪便镜检有大量白细胞，脓细胞和红细胞
	+ 确诊有赖于==粪便培养出痢疾杆菌==

## 慢性菌痢的鉴别诊断
1. 结肠癌与直肠癌
	1. 中老年热常见
	2. 肠道肿瘤继发感染时症状与菌痢相似，抗菌药物可暂缓解症状
	3. 久治无效，伴进行性消瘦
	4. 需肛诊、肠镜和病理学检查确诊
2. [[正番内容/内科学/Episode 03. 消化系统/溃疡性结肠炎\|溃疡性结肠炎]]
	1. 病程长，有脓血便，抗生素治疗无效
	2. 大便培养：阴性
	3. 结肠镜检查：肠粘膜充血、水肿、溃疡形成，黏膜脆弱易出血；晚期患者钡剂灌肠，见结肠袋消失，呈铅管样改变
# 治疗
## 急性菌痢
1. 一般治疗：
	+ ==隔离==：消化道隔离至临床症状消失，大便培养连续两次阴性
	+ **饮食**：继续饮食
2. 病原治疗：
	+ ==喹诺酮类：首选==
	+ 头孢曲松：二线
	+ 黄连素：减少肠道分泌作用
3. 对症治疗：
	+ 补液
	+ 高热：降温
	+ 腹痛
>[!attention]
>+ 儿童、孕妇、哺乳期妇女慎用==喹诺酮类抗生素==
>+ ==口服抗生素吸收好==，不能口服者静脉滴注
## 慢性菌痢
1. 一般治疗
2. 病原治疗：
	+ 根据药敏试验，选择抗菌药物，可以联用
	+ 药物灌肠治疗
	+ 菌群失调者可用益生菌
3. 对症治疗：肠功能紊乱者可镇静或解痉药物
## 中毒性菌痢
1. 一般治疗：密切观察意识状态和生命体征
2. 病原治疗：
	+ ==氟喹诺酮类、三代头孢菌素类==
	+ 抗生素静滴，病情好转改口服
3. 对症治疗：
	+ 降温镇静
	+ 休克型：抗休克治疗-->扩容、纠酸、改善微循环、应用[[未发售DLC/糖皮质激素\|糖皮质激素]]
	+ 脑型：减轻脑水肿、防止呼衰
# 预防
+ 管理传染源：病人和带菌者==隔离、报告==
+ 切断传播途径
	+ ==关键措施==：粪口传播
	+ ==三管一灭==：粪便管理、水源管理、饮食卫生管理和消灭苍蝇
+ 保护易感人群：5 y 以下儿童
